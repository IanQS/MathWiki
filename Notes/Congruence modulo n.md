``` ad-Definition
title: Definition.

_Fix $n\in\N^+$. Define the equivalence relation $\mod{n}$ on $\Z$, called **congruence modulo $n$**, by_
$$\begin{equation}
    a\mod{n}b\ \ \ \ \ \ \ \ \,\colon\!\Leftrightarrow\ \ \ \ \ \ \ \ n\divides\!\l(a-b\r)
\end{equation}$$
_for all $a,b\in\Z$._

```
