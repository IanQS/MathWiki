``` ad-Proposition
title: Proposition.

_Let $G$ be a group. Then, for all $g\in G$, we have $\ord{g}=\ord{\l\langle g\r\rangle}$._

```

**Remark.** In particular, if $G=\l\langle x\r\rangle$ for some $x\in G$, then $\ord{x}=\ord{G}$.<span style="float:right;">$\blacklozenge$</span>
