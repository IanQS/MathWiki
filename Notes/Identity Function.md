``` ad-Definition
title: Definition.

_Let $X$ be a set. The **identity function on $X$** is the function_
$$\begin{equation}
    \id_X\!:X\to X\ \ \ \ \ \ \ \ \textrm{\it{mapping}}\ \ \ \ \ \ \ \ x\mapsto x.
\end{equation}$$
_Formally,_
$$\begin{equation}
    \id_X\!\coloneqq\l\{z\mid\ex x\in X:z=\tpl{x,x}\r\}.
\end{equation}$$
```
