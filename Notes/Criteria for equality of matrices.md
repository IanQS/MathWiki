``` ad-Proposition
title: Proposition.

_Let $K$ be a field and fix $m,n\in\N$. Then, for all $A,B\in\mat{m\times n}{K}$ and all $\v{x}\in K^n$,_
$$\begin{equation}
    A\v{x}=B\v{x}\ \ \ \ \Rightarrow\ \ \ \ A=B.
\end{equation}$$

```

_Proof_.
