``` ad-Definition
title: Definition.

_Let $X$ and $Y$ be sets. A function $f:X\to Y$ is said to be **right-invertible** if there exists a function $g:Y\to X$ such that $f\circ g=\id_X$, in which case $g$ is called a **right-inverse of $f$**._

```
