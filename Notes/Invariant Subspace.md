``` ad-Definition
title: Definition.

_Let $V$ be a vector space over some field $K$ and let $T:V\to V$. A linear subspace $U\subseteq V$ is called **$T$-invariant** if $\l.T\r|_U$ closes in $U$; that is, if_
$$\begin{equation}
    u\in U\ \ \ \ \Rightarrow\ \ \ \ T\l(u\r)\in U.
\end{equation}$$

```
