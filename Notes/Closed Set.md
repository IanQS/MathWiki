``` ad-Definition
title: Definition.

_Let $\tpl{X,\mc{T}}$ be a topological space. A subset $C\subseteq X$ is said to be a **closed set of $X$ w.r.t. $\mc{T}$** if $X\comp C$ is open in $\mc{T}$._
```
