``` ad-Definition
title: Definition.

_Let $u$ be a set and let $z$ be any set asserted by $\axipow$. The **power set of $u$** is the set_
$$\begin{equation}
    \pow u\coloneqq\l\{x\in z\mid x\subseteq u\r\}.
\end{equation}$$
```
