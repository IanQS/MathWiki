``` ad-Proposition
title: Proposition.

_Let $G$ be a finite group and fix $g\in G$. Then $g^\ord{G}=e$._

```

_Proof_. By Lagrange$\textrm{'}$s Theorem, $\ord{g}$ divides $\ord{G}$. The result follows since
$$\begin{equation}
    g^\ord{G}=e\ \ \ \ \ \ \ \ \Leftrightarrow\ \ \ \ \ \ \ \ \ord{g}\textrm{ divides }\ord{G}.\qedin
\end{equation}$$
