``` ad-Proposition
title: Proposition.

_Let $V,W,Z$ be finite-dimensional vector spaces over some field $K$ with bases_
$$\begin{equation}
    \mc{B}\coloneqq\l\{b_1,\dots,b_n\r\},\ \ \ \ \ \ \ \ \mc{C}\coloneqq\l\{c_1,\dots,c_m\r\}, \ \ \ \ \textrm{\it{and}}\ \ \ \  \mc{D}\coloneqq\l\{d_1,\dots,d_p\r\},
\end{equation}$$
_respectively, and consider any linear maps $T:V\to W$ and $U:W\to Z$. Then_
$$\begin{equation}
    \l[U\circ T\r]_\mc{B}^\mc{D}=\l[U\r]_\mc{C}^\mc{D}\l[T\r]_\mc{B}^\mc{C}.
\end{equation}$$

```
