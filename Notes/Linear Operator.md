``` ad-Definition
_Let $V$ be a $K$-vector space. A linear map $T:V\to V$ is said to be a **linear operator**._

```

**Remark.** The set of all linear operators $T:V\to V$ is denoted $\edm\l(V\r)$.<span style="float:right;">$\blacklozenge$</span>
