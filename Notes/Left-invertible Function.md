``` ad-Definition
title: Definition.

_Let $X$ and $Y$ be sets. A function $f:X\to Y$ is said to be **left-invertible** if there exists a function $g:Y\to X$ such that $g\circ f=\id_X$, in which case $g$ is called a **left-inverse of $f$**._

```
