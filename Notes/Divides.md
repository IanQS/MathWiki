title: Definition.

_Let $a\in\Z$ and $b\in\Z\comp\l\{0\r\}$. The **divisibility relation on $\Z$** is the relation $\divides\!$ defined by_
$$\begin{equation}
    b\divides a\ \ \ \ \ \ \ \ \,\colon\!\Leftrightarrow\ \ \ \ \ \ \ \ \ex k\in\Z:a=kb.
\end{equation}$$
