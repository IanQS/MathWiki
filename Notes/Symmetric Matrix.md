``` ad-Definition
title: Definition.

_Let $K$ be a field and fix $m,n\in\N$. A matrix $A\in\mat{m\times n}{K}$ is said to be **symmetric** if $A=A^\trans$._

```

**Remark.** Symmetric matrices are necessarily square, so $m=n$.<span style="float:right;">$\blacklozenge$</span>
