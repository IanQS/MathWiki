``` ad-Definition
title: Definition.

Let $\l\{X_i\r\}_{i\in I}$ be an indexed family. The <b>projection map onto the $j^\textrm{\it{th}}$ coordinate</b> is the function
$$\begin{equation}
    \pi_j:\prod\limits_{i\in I}X_i\to X_j\ \ \ \ \mathit{mapping}\ \ \ \ f\mapsto f\l(j\r).
\end{equation}$$
```
