!!! ad-Definition Definition.

Let $X$ be a set. The **set of topologies on $X$** is the set
$$\begin{equation}
    \Sigma\coloneqq\l\{\mc{T}\in\mc{P}\l(\mc{P}\l(X\r)\r)\mid\mc{T}\textrm{ is a topology on }X\r\}.
\end{equation}$$
