!!! ad-Definition Definition.

Let $X$ be a set and let $\sim$ be an equivalence relation on $X$. The **quotient set of $X$ under $\sim$** is the set
$$\begin{equation}
    X/\!\sim\,\coloneqq\l\{\l[x\r]_\sim\in\mc{P}\l(X\r)\mid x\in X\r\}.
\end{equation}$$
