!!! ad-Definition Definition.

Let $X$ be a set. The **identity function on $X$** is the function
$$\begin{equation}
    \id_X\!:X\to X\ \ \ \ \ \ \ \ \textrm{mapping}\ \ \ \ \ \ \ \ x\mapsto x.
\end{equation}$$
Formally,
$$\begin{equation}
    \id_X\!\coloneqq\l\{z\mid\ex x\in X:z=\l\langle x,x\r\rangle\r\}.
\end{equation}$$
