title: Definition.

_Let $u$ be a set and let $z$ be any set asserted by the Axiom of Power Set. The **power set of $u$** is the set_
$$\begin{equation}
    \mc{P}u\coloneqq\l\{x\in z\mid x\subseteq u\r\}.
\end{equation}$$
